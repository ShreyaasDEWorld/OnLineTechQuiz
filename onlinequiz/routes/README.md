# Elearning Platform (Flask)

A full-featured e-learning platform built with Flask and PostgreSQL.

## Features
- User Signup with Email OTP Verification
- Secure Login (Flask-Login)
- Admin Quiz Management
- Timed Quiz System
- Score Tracking
- Password Reset via OTP

## Tech Stack
- Python 3.12
- Flask
- PostgreSQL
- Flask-Login
- Psycopg2

## Setup Instructions

```bash
git clone https://github.com/yourusername/elearning.git
cd elearning
pip install -r requirements.txt
python app.py
